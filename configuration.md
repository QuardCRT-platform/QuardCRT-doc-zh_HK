# 配置

quardCRT 有兩層配置：

- 全域配置：對整個應用程式生效的預設值與偏好設定。
- 工作階段配置：只影響目前工作階段或某個已儲存連線的設定。

實際使用時，通常先透過全域配置設定外觀與工作流程預設值，再在特定目標需要不同行為時，以工作階段配置個別覆蓋。

## 全域配置

全域配置檔通常位於使用者目錄下，例如 `<使用者目錄>/.config/QuardCRT/QuardCRT.ini`。如果你想確認目前機器實際使用的路徑，最可靠的方法是開啟 `選項 > 全域選項 > 高級`，查看其中的 `配置檔案路徑` 欄位。

![配置檔案所在路徑](./img/configuration_1.png)

對於已儲存密碼等敏感資訊，quardCRT 不會將明文密碼直接寫入這個配置檔，而是依賴系統鑰匙圈服務。詳情可參考 [常見問題](./faq.md)。

## 如何選擇全域設定與工作階段設定

- 主題、字型、回捲緩衝區、傳輸目錄、預設視窗行為等希望在所有分頁中統一生效的項目，請使用全域設定。
- 協議、連接埠、鮑率、驗證方式、終端行為等只針對某一台主機或裝置的項目，請使用工作階段設定。
- 如果不確定，優先修改全域設定；只有在確實需要時，再於工作階段層級覆蓋。

## 建議優先查看的部分

多數新使用者通常會先看以下幾個區塊：

- `外觀`：選擇主題、字型與可選的背景媒體。
- `終端`：設定回捲緩衝區大小、游標行為與單字選取行為。
- `傳輸`：設定上傳與下載目錄。
- `高級`：確認配置檔案路徑與可選的介面整合功能。

以下將依分類介紹全域配置的各個部分。

### 常規

常規控制整個應用中新增分頁與預覽的行為。

- 新分頁模式

	新分頁模式表示當使用者使用分頁 ➕ 按鈕或快捷鍵新增分頁時，新分頁的開啟方式。有三種模式可選：

	- 新工作階段：彈出快速連線對話框，使用者配置新分頁的連線資訊，產生一個新的工作階段。
	- 複製工作階段：複製目前分頁的工作階段資訊，產生一個新的工作階段（如果目前分頁硬體資源為單一裝置，會彈出快速連線對話框，讓使用者配置不可複製的硬體資源資訊）。
	- 本地終端：產生一個新的本地終端工作階段。

- 新分頁工作目錄

	當使用者建立一個新的本地終端工作階段時，新工作階段的工作目錄。預設情況下該選項僅包含一個選項：`使用者的主目錄`，如需加入其他選項，需先將其加入目錄書籤。

- 分頁標題模式

	分頁標題模式表示分頁標題的顯示方式。有三種模式可選：

	- 簡要：固定寬度顯示，不顯示完整的工作階段標題。
	- 完整：顯示完整的工作階段標題。
	- 捲動：顯示完整的工作階段標題，當標題過長時會自動捲動顯示。

- 分頁標題寬度

	當分頁標題模式為簡要或捲動時，分頁標題的固定寬度。單位為像素。

- 分頁預覽

	當使用者滑鼠懸停在分頁上時，顯示分頁預覽資訊。

- 預覽視窗寬度

	分頁預覽視窗的寬度。單位為像素。

### 外觀

外觀控制終端的視覺樣式。

- 配色方案

	quardCRT 提供數十種配色方案供使用者選擇，你可以依照偏好選擇合適的配色。

- 字型

	終端的字型設定包含字型名稱與字型大小。其中 Built-in 表示 quardCRT 內建字型，使用者也可以選擇系統中已安裝的字型。Built-in 字型選用了適合程式開發的等寬字型；如果要將已選字型改回 Built-in，可在選擇字型時點擊取消按鈕恢復。

- 背景圖片

	你可以選擇圖片作為終端背景。支援的圖片格式包括 `bmp`、`jpg`、`jpeg`、`png`、`gif`。GIF 會以動畫背景顯示。如果在 `高級` 中啟用了 `終端背景支援動畫`，還可以選擇 `mp4`、`avi`、`mov`、`mkv` 等影片格式，但這可能會明顯增加資源消耗。點擊清除按鈕可移除目前背景。

- 背景模式

	終端背景模式有六種：

	- 無：不調整背景圖片大小。
	- 拉伸：拉伸背景圖片以適應終端大小。
	- 縮放：縮放背景圖片以適應終端大小。
	- 適應：保持背景圖片長寬比，使其完整顯示在終端中。
	- 置中：將背景圖片置中顯示在終端中。
	- 平鋪：以平鋪方式填滿整個終端。

- 背景圖片透明度

	終端背景透明度，取值範圍為 0-100，0 表示完全透明，100 表示完全不透明。

### 終端

終端包含終端元件本身的核心行為設定。

- 回捲行數

	終端的回捲行數表示終端中最多快取的行數。較大的值會占用更多記憶體。

- 游標樣式

	終端游標樣式有三種可選：

	- 區塊：顯示為實心矩形區塊。
	- 底線：顯示為底線。
	- I 形線：顯示為垂直線。

- 游標閃爍

	勾選後游標會閃爍。

- 單字字元

	表示將哪些特殊字元視為單字的一部分，用於雙擊選取。預設為：`@-./_~`。

### 視窗

視窗包含應用層級的視窗行為設定。

- 視窗透明度

	整個應用程式視窗透明度，取值範圍為 0-100，0 表示完全透明，100 表示完全不透明。

### 傳輸

傳輸用於設定內建檔案傳輸功能的預設路徑與行為。

- 上傳目錄

	當使用者在工作階段中上傳檔案時，檔案預設上傳到的目錄。

- 下載目錄

	當使用者在工作階段中下載檔案時，檔案預設下載到的目錄。

- X/Ymodem 傳送資料封包大小

	X/Ymodem 協議傳送資料封包的大小。預設為 128 位元組。

- Zmodem 線上

	勾選表示 Zmodem 協議處於線上狀態，此時終端工作階段可即時偵測 Zmodem 檔案傳輸請求，取消勾選則不再即時回應。

### 高級

高級包含配置檔案路徑顯示，以及可能影響整合方式或效能的可選功能。

- 配置檔案路徑

	quardCRT 的配置檔案路徑，使用者可在此查看目前配置檔所在位置。

- 翻譯服務

	quardCRT 的翻譯服務，使用者可選擇 Google 翻譯、百度翻譯或 Bing 翻譯。

- 終端背景支援動畫

	勾選表示終端背景支援動畫，使用者可選擇 mp4、avi、mov、wkv 等影片格式作為終端背景，但這會消耗較多系統資源。

- 原生 UI

	勾選表示使用原生 UI，主要供 macOS 使用者使用。啟用後，quardCRT 介面會更貼近 macOS 原生風格。若你經常在多個作業系統間切換，通常更適合保持關閉，以維持一致的跨平台體驗。修改後需重新啟動 quardCRT 才會生效。

- Github Copilot

	此功能預留給未來版本，目前尚不可用。

## 工作階段配置

當某個連線需要與全域預設值不同的設定時，工作階段配置就很有用。典型例子包括：

- 序列裝置需要特定鮑率與流控模式
- SSH 伺服器需要金鑰驗證而非密碼驗證
- 某台主機應始終以特定名稱、協議或連接埠開啟

如果你會反覆使用同一個目標，建議在確認設定後將其儲存為工作階段。
